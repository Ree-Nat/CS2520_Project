# CS2520_Project - NYT Games Clone

A web-based collection of word and logic games inspired by the New York Times Games, featuring a cohesive design system with glassmorphism effects and smooth micro-interactions.

## Features

### Available Games
- **Wordle** - Guess the five-letter word in 6 tries with color-coded feedback
- **Spelling Bee** - Form words using the given letters to reach target scores
- **Tic-Tac-Toe** - Classic strategy game with a modern, animated interface

### Design System
The application features a unified design language across all games:
- **Glassmorphism Effects** - Semi-transparent backgrounds with blur for hero elements
- **Micro-interactions** - Smooth hover effects, scale transforms, and ripple animations
- **Lime-Green Theme** - Consistent accent color (#8aba18) throughout
- **Responsive Layout** - Clean, centered layouts optimized for gameplay

## Setup

```bash
pip install -r requirements.txt
python app.py
```

Visit `http://localhost:5000` in your browser to play.

## Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Architecture**: MVC pattern with session-based state management
- **Styling**: Custom CSS with glassmorphism and animations
